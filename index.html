About this experiment

An experiment to prove the formula: Minimalist Harness + Security Hardening + Open Small Models + Enterprise Skills = Enterprise AI.

Built on rh-agent, a rootless container that pairs open-weight models with curated enterprise skills. This project pushes the barrier further — everything runs entirely in the browser via WebGPU. No server, no container runtime, no API keys.
